Projeto feito com spring boot com integração ao banco de dados PostgreSQL (eu não tinha MySql na minha máquina), meu projeto foi dividido em "entities" que eram as classes solicitadas Tarefas e Itens com seus devidos atributos. O "repository" uma interface que herda do JpaRepository, quando quero que uma classe trabalhe com banco de dados é bom utilizar esse repositório para trabalhar com dados. O service é onde eu controlo as regras de negócio e meus metodos e o controller para criar meus endpoints e mapeamento do verbos htpp. Obrigado pela oportunidade!
